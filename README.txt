HOW TO LOAD WEBSITE

1) Ensure all files are included in folder CARROOM, with app folder, public folder, and env.json.

2) Visit https://api-ninjas.com/ and create a free account. 

3) Locate your unique API key under "My Account", and click "Show API Key" button.

4) Plug in your unique API key in the env.json file beside "api_key" replacing "YOUR_API_KEY_HERE".

5) Cd into the CARROOM directory and run "npm i".

6) Run "npm run start" and then visit localhost:3000 on the browser
