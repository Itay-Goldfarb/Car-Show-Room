HOW TO LOAD WEBSITE

1) Ensure all files are included in folder CARROOM, with app folder, public folder, and env.json.

2) Plug in your unique API key for API-Ninjas in the env.json file beside "api_key1" replacing "API_NINJAS_API_KEY_HERE".

3) Plug in your unique API key for Pexels in the env.json file beside "api_key2" replacing "PEXELS_API_KEY_HERE".

4) Cd into the CARROOM working directory and run "npm i".

5) Run "npm run start" and then visit localhost:3000/home.html on the browser
